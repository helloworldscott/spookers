# The Lighthouse Keeper

Low-poly 3D horror survival game built with vanilla HTML/CSS/JS + Three.js CDN.

## Run locally
- Open `index.html` directly in a modern browser, or
- Serve folder with a static server (recommended for module loading):
  - `python3 -m http.server 8080`
  - visit `http://localhost:8080`

## Deploy to GitHub Pages
1. Push this folder to a GitHub repo.
2. In repo settings, enable **Pages** and set source branch (e.g. `main` root).
3. Open the published URL.

No external assets are required.
